#
# Each line maps one XDG icon name to the closest Lucide icon. Coverage
# targets the Icon Naming Specification (0.8.90) plus the de-facto names
# shipped by Adwaita that GTK apps rely on (open-menu, pan-*, checkbox-*,
# power-profile-*, ...). The keywork icon theme ships every Lucide SVG
# under its original name; build/gen_icon_theme.zig turns each line here
# into a symlink `<xdg-name>.svg -> <lucide-name>.svg`.
#
# Names identical in both sets (folder, battery, printer, ...) need no
# entry; the generator rejects aliases that collide with a Lucide name.
